# Winner Partner

Winner Partner是一个围棋辅助程序，用于检测屏幕上特定区域的深绿色像素点，并将其转换为围棋坐标。

## 功能特点

### 核心算法
1. **全局遮罩极速扫描**：使用 OpenCV 全局过滤替代局部轮询，性能提升约20倍（从 ~100ms 降至 ~5ms）
2. **动态阈值计算**：根据棋盘大小自动调整扫描范围和颜色阈值，适配不同分辨率
3. **防漏判机制**：无状态扫描设计，每次扫描独立判断，避免错过重复落子（提子后再落子）
4. **二次验证逻辑**：向上逐个像素检测，确认绿色标记点的真实性，降低误判率

### 自动化与快捷键
5. **自动监控模式**：每 300ms 高频扫描，实时捕获对手落子
6. **全局快捷键支持**：
   - `F9` - 强制扫描 Override（即时触发一次扫描）
   - `F10` - 开启/停止自动监控

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python src/main.py
```

### 使用流程

1. **设置棋盘区域**：点击“重新确定位置”，用鼠标框选棋盘最外侧黑线围成的矩形
2. **开启自动监控**：点击“开启自动监控 (F10)”或按 `F10` 键，程序开始每 300ms 扫描一次
3. **实时捕获落子**：当对手落子时，绿色标记点会被自动识别并显示在界面上
4. **强制扫描**：如需即时扫描，按 `F9` 键触发 Override
5. **手动下棋**：在输入框中输入坐标（如 a15），点击“下棋”按钮
6. **结束程序**：点击“结束”按钮，日志会自动保存为带时间戳的文件

## 重要提示

### 棋盘框选精度要求

**请将红色矩形框精准对齐棋盘最外侧的黑线（第1线和第19线）**

- ✅ 正确做法：框的边缘紧贴棋盘最外围的四条线
- ❌ 错误做法：框包含棋盘外的木质边缘或留白区域

**原因**：程序会将红色矩形按 18 等分计算交叉点坐标。如果框选不准确，所有坐标都会错位，导致识别失败。

### 日志文件说明

每次运行程序时，棋谱记录会自动保存为带时间戳的文件（如 `mygame_20260504_143025.txt`），避免覆盖历史对局记录。
