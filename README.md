# Winner Partner

Winner Partner是一个围棋辅助程序，用于检测屏幕上特定区域的深绿色像素点，并将其转换为围棋坐标。

## 功能特点

1. 扫描检测像素点为深绿色
2. 检测到后并不跳出，继续扫描检测
3. 如果棋盘上有多个深绿色，将他们的围棋坐标一一指出来
4. 对这些深绿色的屏幕坐标点进行二次筛选：
   - 从该屏幕坐标点向上逐个像素检测
   - 如果是绿色像素点，则继续向上检测
   - 如果遇到白色像素点，则标记最初的像素点为正确
   - 如果显示黑色像素点，则继续向上检测，直到出现不同颜色的像素点
   - 如果最先出现的是白色像素点，则标记正确并将坐标转换为围棋坐标显示
   - 否则标记最初的像素点为错误

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python src/main.py
```
