# Winner Partner

Winner Partner是一个围棋辅助程序，用于检测屏幕上特定区域的深绿色像素点，并将其转换为围棋坐标。

## 功能特点

1. 扫描检测像素点为深绿色
2. 检测到后并不跳出，继续扫描检测
3. 如果棋盘上有多个深绿色，将他们的围棋坐标一一指出来
4. 对这些深绿色的屏幕坐标点进行二次筛选：
   - 从该屏幕坐标点向上逐个像素检测
   - 如果是绿色像素点，则继续向上检测
   - 如果遇到白色像素点，则标记最初的像素点为正确
   - 如果显示黑色像素点，则继续向上检测，直到出现不同颜色的像素点
   - 如果最先出现的是白色像素点，则标记正确并将坐标转换为围棋坐标显示
   - 否则标记最初的像素点为错误
5. **动态阈值计算**：根据棋盘大小自动调整扫描范围和颜色阈值，适配不同分辨率
6. **防漏判机制**：每次扫描前清空缓存，避免错过重复落子

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python src/main.py
```

## 重要提示

### 棋盘框选精度要求

**请将红色矩形框精准对齐棋盘最外侧的黑线（第1线和第19线）**

- ✅ 正确做法：框的边缘紧贴棋盘最外围的四条线
- ❌ 错误做法：框包含棋盘外的木质边缘或留白区域

**原因**：程序会将红色矩形按 18 等分计算交叉点坐标。如果框选不准确，所有坐标都会错位，导致识别失败。

### 日志文件说明

每次运行程序时，棋谱记录会自动保存为带时间戳的文件（如 `mygame_20260504_143025.txt`），避免覆盖历史对局记录。
